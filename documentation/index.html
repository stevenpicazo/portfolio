Installation

Follow the steps below to setup your site template:

  1. NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

    You'll need to install Node.js >=v17.0+ (Recommended Version)(NPM comes along with it) and serve (a NPM package)

  2. Let's see how the setup works:
    • Use terminal and navigate to the project root.
    • Then run :npm install
    • Then run :npm run start
    • Now in the browser go localhost:3000
  3. You're now good to go..!

Changing Fonts

Simone template use fonts: Poppins from the Google Fonts Library. You can change the fonts from page head in HTML files (as following code).

From the template options go to public > index.html

<!-- Web Fonts
============================================= -->
<link rel='stylesheet' href='https://fonts.googleapis.com/css2?family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap' >
  • In order to change the Fonts, you will need to Edit the Above Links with your Chosen Google Fonts.
  • Then Replace the body-font variable value in the src/sass/_variables.scss File with your Newly Added Fonts.

Sass

We have added SASS .scss files in template. If you know how to use SASS you can change sass files and compile the css as well. You can find sass file here - src/sass

Open the src/sass/_variables.scss and Edit the values according to your needs. If you need more Advanced Setup then you can Edit the Respective Files yourself which have been branched inside the same Folder. It is completely at your discretion only to include the Required .scss Files you need to minimize the amount of CSS & including only the Styles of the Blocks you need. This can be setup in your stylesheet.scss File.

Setting up RTL

We have included all the Tools to make your Website completely RTL ready. Simply follow the steps below:

Code Example:

  • Step 1

    Add the dir="rtl" to the <html> element at public/index.html file.
    Add an appropriate lang attribute, like lang="ar", on the <html> element.

    Example:

    <html dir="rtl" lang="ar">

  • Step 2

    From src/index.js, you’ll need to apply an RTL version of Bootstrap CSS instead of default bootstrap css.
    For that :
    comment out import "bootstrap/dist/css/bootstrap.min.css";
    and uncomment // import "bootstrap/dist/css/bootstrap.rtl.min.css";
